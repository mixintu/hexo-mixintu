Updated on 2020-04-21

java中==和equals的区别是什么?
equals 方法是用于比较两个独立对象的内容是否相同，就好比去比较两个人的长相是否相同，它比较
的两个对象是独立的。
要比较两个基本类型的数据或两个引用变量是否相等，只能用==操作符。
字符串的比较基本上都是使用 equals 方法
